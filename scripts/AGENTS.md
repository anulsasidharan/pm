# Scripts Agent Guide

This folder contains start and stop scripts for Windows, macOS, and Linux.

## Responsibilities

- Provide simple commands to start and stop the local Docker stack
- Keep scripts readable and minimal
- Ensure scripts use `docker compose` from repository root

## Files

- `start-windows.bat`
- `stop-windows.bat`
- `start-mac.sh`
- `stop-mac.sh`
- `start-linux.sh`
- `stop-linux.sh`
- `verify-part2-windows.bat`
- `verify-part2-mac.sh`
- `verify-part2-linux.sh`
- `verify-part3-windows.bat`
- `verify-part3-mac.sh`
- `verify-part3-linux.sh`
